# CatalogoApp

Este proyecto tiene el objetivo de hacer un catalogo como la anterior vez para la implementacion de algo nuevo, el cual la reseña, y dejar una opinion ademas de poder realizar un inicio de sesion para que puedas dejar tu reseña y se guarde acumulandose las reseñas.

# Tecnologías Utilizadas

Backend: C# / .NET
Framework CSS: Bootstrap 5
Tipografía: Orbitron (Google Fonts)
Herramientas: Visual Studio 2022, Git
